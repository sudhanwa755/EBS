# Electricity Billing System (Frontend)

A modern, responsive frontend for an Electricity Billing System built with HTML, CSS (Tailwind), and Vanilla JavaScript.

## Features

- **Role-Based Access Control**: Admin and User portals.
- **Dashboard**: Visual analytics using Chart.js.
- **Bill Management**: View, download, and pay bills.
- **Admin Tools**: Manage users, tariffs, and meter readings.
- **Mock Backend**: Simulates API calls using local data.

## Tech Stack

- HTML5
- CSS3 (Tailwind CSS via CDN)
- Vanilla JavaScript (ES6 Modules)
- Chart.js
